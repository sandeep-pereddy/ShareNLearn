/********************************************************************
	Execution instructions
********************************************************************/

1. Install Node.js using https://nodejs.org/en/.

2. Install MQSQL server.

3. Install Spring Tool Suite4.

4. Change mysql username and password in application.properties file present in the Java project.

5. Run Spring Boot Application.

6. Run React project using the commands mentioned below.


/********************************************************************
	Commands to Run React Project
********************************************************************/

1. npm install - To install missing libraries.
2. npm start to start the angular application.
